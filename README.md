# PedestrianDetection
行人检测系统，探测车前的行人，如果有人走入汽车的行进路线即会发出警告


运行环境: Python3.6(Miniconda3)+OpenCV3，IDE：PyCharm

环境安装：
pip install -r requirements.txt
